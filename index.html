1.How does javascript work?

যখন আমরা ব্রাউজারে js file চালাই তখন chrome er V8 engine js code টাকে receive করে । parser এর মাধ্যমে একে parse করা হয় । next step এ engine একটা Abstract syntax tree তৈরি করে । এরপর কোডগুলোকে interpret করা শুরু হয় (যাকে বলে interpreter ignition) যা কোডগুলোকে byte code এ নেয় । এছাড়াও V8 engine এ JIT কম্পাইলার থাকে যা কোডগুলোকে compile করে দেখে কত দ্রুত আউটপুট দেয়া যায়। পাশাপাশি optimizing compiler(turbofan) থাকে যা optimized machine code দেয়। finally program টি execute হয়।


2.Local Storage vs Session Storage

Local storage এবং session storage হল দুটি Web storage API. এই API গুলো আমাদের browser এ data read এবং write করার সুবিধা দিয়ে থাকে । এদের মধ্যে main differance হল local Storage এ কোন data store করে থাকলে তা হবে permanent, browser close করলে বা manually delete না করলে তা remove হবেনা, কিন্ত session storage এ data শুধু একটা নির্দিষ্ট session এর জন্য stored থাকে,browser close করলে তা remove হয়ে যাবে । Local storage ১০এমবি আর session storage ৫এমবি storage নিয়ে কাজ করে। Local storage যে কোন window থেকে accessible, session storage শুধু same tab এ access করা যায়।
